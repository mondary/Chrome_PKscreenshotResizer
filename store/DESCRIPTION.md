# PK Screenshot Resizer - Description Chrome Web Store

## 🇬🇧 English

Resize windows & capture screenshots. Made by PK-Labs.

Powerful browser window manager and screenshot tool. Resize windows to standard dimensions (mobile, tablet, desktop) and capture high-quality screenshots.

**Features:**
- Resize browser windows to standard sizes
- Mobile (375x667), Tablet (768x1024), Desktop (1920x1080) presets
- Custom dimensions support
- High-quality screenshot capture
- Multiple format options
- Perfect for developers and designers

Perfect for web developers, UI/UX designers, content creators, and anyone who needs consistent window sizes and screenshots.
