Politique de confidentialité

PK Screenshot Resizer Chrome Extension

Dernière mise à jour : 26 juin 2026

Cette page est la politique de confidentialité officielle de l'extension Chrome PK Screenshot Resizer. This page is the official privacy policy for the PK Screenshot Resizer Chrome extension. An English version is available below the French version.

Données traitées

PK Screenshot Resizer ne collecte pas de données personnelles pour un serveur externe. L'extension traite uniquement les données nécessaires pour redimensionner la fenêtre active, capturer un screenshot et enregistrer ce fichier localement :

  • L'onglet actif et sa fenêtre pour effectuer la capture ;
  • L'image capturée générée localement par Chrome ;
  • Le nom du fichier téléchargé par l'utilisateur.

Utilisation des données

Ces données sont utilisées uniquement pour effectuer la capture d'écran demandée par l'utilisateur et lancer son téléchargement local.

Stockage et transfert

L'extension n'envoie pas les captures ni les informations de navigation à un serveur externe. Les fichiers sont générés localement par Chrome puis téléchargés sur l'appareil de l'utilisateur via l'API de téléchargement du navigateur.

Partage et transfert

PK Screenshot Resizer ne vend pas, ne loue pas, ne partage pas et ne transfère pas les données utilisateur à des tiers. Les données ne sont pas utilisées pour la publicité, le suivi analytique, le profilage, la solvabilité ou des finalités sans rapport avec la fonctionnalité de base de l'extension.

Cookies et suivi

L'extension n'utilise pas de cookies, de pixels de suivi, d'outils analytiques ou de technologies publicitaires.

Accès et suppression

Les captures sont enregistrées uniquement si l'utilisateur confirme le téléchargement. Les fichiers peuvent être supprimés à tout moment depuis l'ordinateur de l'utilisateur. L'extension ne conserve pas d'historique externe des captures.

Permissions Chrome

Les permissions déclarées servent uniquement à identifier l'onglet actif, redimensionner la fenêtre courante, capturer la partie visible et sauvegarder le screenshot sur l'appareil de l'utilisateur.

Modifications de cette politique

Cette politique peut être mise à jour si les fonctionnalités de l'extension changent. La date de mise à jour sera modifiée sur cette page.

Contact

Pour toute question concernant cette politique de confidentialité, contactez : contact@mondary.design.

English Version

Data Processed

PK Screenshot Resizer does not collect personal data for an external server. The extension only processes the information required to resize the active window, capture a screenshot, and save that file locally:

  • The active tab and its window in order to perform the capture;
  • The screenshot image generated locally by Chrome;
  • The filename downloaded by the user.

Use of Data

This data is used only to perform the screenshot requested by the user and start its local download.

Storage and Transfer

The extension does not send screenshots or browsing information to an external server. Files are generated locally by Chrome and then downloaded to the user's device through the browser download API.

Sharing and Transfer

PK Screenshot Resizer does not sell, rent, share, or transfer user data to third parties. Data is not used for advertising, analytics tracking, profiling, creditworthiness, or purposes unrelated to the extension's core functionality.

Cookies and Tracking

The extension does not use cookies, tracking pixels, analytics tools, or advertising technologies.

Access and Deletion

Screenshots are saved only if the user confirms the download. Files can be deleted at any time from the user's computer. The extension does not keep an external history of captures.

Chrome Permissions

The declared permissions are used only to identify the active tab, resize the current window, capture the visible area, and save the screenshot on the user's device.

Changes to This Policy

This policy may be updated if the extension features change. The update date will be changed on this page.

Contact

For questions about this privacy policy, contact: contact@mondary.design.
