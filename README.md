# Screenshot Extension 1200x1200

![Project icon](icon.png)

[🇫🇷 FR](README.md) · [🇬🇧 EN](README_en.md)

✨ Extension Chrome pour redimensionner la fenêtre à 1200x1200 pixels et capturer des screenshots.

## ✅ Fonctionnalités
- Sélection de multiple formats prédéfinis pour différentes plateformes
- Redimensionnement automatique de la fenêtre selon le format sélectionné
- Capture de screenshot avec redimensionnement
- Téléchargement automatique du fichier avec un timestamp
- Nommage intelligent du fichier selon la plateforme
- Interface simple avec sélecteur de format et bouton d'action
- Status temps réel de l'opération

## 🧠 Utilisation
1. Cliquez sur l'icône de l'extension dans la barre d'outils Chrome
2. Sélectionnez le format désiré dans le menu déroulant :
   - Chrome Store (1200x1200)
   - Android Store (1024x1024)
   - iOS App (1024x768)
   - macOS App (512x512)
   - WordPress (1200x630)
   - VSCode Marketplace (1280x720)
3. Cliquez sur le bouton "Capturer"
4. La fenêtre sera redimensionnée automatiquement
5. Le screenshot sera capturé et téléchargé automatiquement

## ⚙️ Réglages
Aucune configuration nécessaire - l'extension fonctionne directement après installation.

## 🧾 Commandes
- Capturer 1200x1200 : Redimensionne la fenêtre et capture un screenshot

## 📦 Build & Package
Extension Chrome utilisant Manifest V3. Aucun build nécessaire - utilisez le dossier directement.

## 🧪 Installation (Antigravity)
1. Ouvrez Chrome
2. Allez dans `chrome://extensions/`
3. Activez le "Mode développeur" (en haut à droite)
4. Cliquez sur "Charger l'extension non empaquetée"
5. Sélectionnez le dossier `Chrome_PKscreenshotResizer` (avec le dossier src à l'intérieur)
