# PKscreenshotResizer

![Project icon](src/icon.png)

[🇫🇷 FR](README.md) · [🇬🇧 EN](README_en.md)

✨ Extension Chrome pour redimensionner la fenêtre et capturer des screenshots aux formats des stores (Chrome, Android, iOS, etc.).

## ✅ Fonctionnalités
- **Interface Apple-like moderne** : Design épuré, icônes par appareil, boutons ergonomiques.
- **Formats prédéfinis** : Chrome Store, Android, iOS, macOS, WordPress, VSCode.
- **Capture en un clic** : Redimensionnement automatique et téléchargement instantané.
- **Workflow pro** : Optimisé pour la soumission rapide sur les Stores.

## 🧠 Utilisation
1. Cliquez sur l'icône de l'extension.
2. Sélectionnez l'appareil/format dans la liste (icônes disponibles).
3. Cliquez sur le bouton "Capture Screenshot".
4. La fenêtre est automatiquement redimensionnée et le screenshot est téléchargé.

## ⚙️ Réglages
Aucune configuration nécessaire - l'extension est prête à l'emploi.
